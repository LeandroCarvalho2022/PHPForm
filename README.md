## E esta é a página que somente pessoas autorizadas da equipe tem acesso, é um dashboard aonde são retornados os valores do banco de dados e depois são contabilizados e disparados, a tabela possui dois estados o 1° aonde na coluna count é mostrado somente a quantidade de vezes que uma determinada opção do dropdown howheard foi escolhida, e o 2° estado aonde mostra a data no qual foi submetido cada opção, este projeto foi debatido e criado para que a equipe de marketing tenha uma idéia de como está sendo recebido pelo público o seu trabalho.
